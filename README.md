# India Mining Sector Intelligence Dashboard

A strategic analytics dashboard tracking India's mining sector performance from 2000-01 to 2015-16.

## Live App
[View Dashboard](https://india-mining-dashboard-6sskz4hps54atmcnhsra54.streamlit.app/)

## What it does
- Tracks total mining value growth across 15 years (367% increase)
- Compares Fuels vs Metallic mineral sector performance
- Interactive mineral production explorer across 7 key minerals
- Year-over-year growth analysis with anomaly callouts

## Key Insights Surfaced
- 2008-09 commodity supercycle drove a 43% YoY spike — highest in the dataset
- Coal dominates fuel value at 48.2%, signalling energy security concentration risk
- 2015-16 marked the first decline (-1.4%), coinciding with global commodity downturn

## Tech Stack
Python | Pandas | Plotly | Streamlit | data.gov.in
